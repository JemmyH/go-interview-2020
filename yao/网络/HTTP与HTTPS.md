# HTTP与HTTPS

## HTTP

http是一个基于请求与响应模式的，无状态的，应用层协议，常基于tcp的连接方式。缺点：传输速度慢，数据包大；如实现实时交互，服务器性能压力大；数据传输安全性差



## HTTP2.0

在不改动HTTP语义、方法、状态码、URI及首部字段的情况下，大幅度提高了web性能。

- 实现了多路复用
- 使用HPACK算法对头部进行了压缩
